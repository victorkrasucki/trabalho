O Sistema de Nomes de Domínio, mais conhecido pela nomenclatura em Inglês Domain Name System, é um sistema hierárquico e distribuído de gestão de nomes para computadores, serviços ou qualquer máquina conectada à Internet ou a uma rede privada.

MS-DOS é uma acrossemia de MicroSoft Disk Operating System (sistema operacional em disco da Microsoft); é um nome genérico do sistema operacional licenciado pela Microsoft Corporation para uso em vários microcomputadores de diferentes de fabricantes.
